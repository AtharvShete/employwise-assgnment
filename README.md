# EmployWise Assignment

## Getting Started

### Installation

First, install the dependencies:

```bash
# Using npm
npm install

# Using pnpm
pnpm install
```

### Development

Then, run the development server:

```bash
# Using npm
npm run dev

# Using pnpm
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
